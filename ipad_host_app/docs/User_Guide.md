# IPA 上位机程序 - 用户操作指南

## 1. 简介

IPA 上位机程序是基于 IPAd SDK 开发的 Linux 桌面应用程序，提供图形界面用于调用 SDK API 实现 GSMA SGP.32 规范中定义的 IPAd（IoT Profile Assistant Distance）能力。

## 2. 系统要求

- **操作系统**: Ubuntu 18.04+ / Debian 10+
- **内存**: 512MB+
- **存储**: 100MB+ 可用空间
- **依赖库**: GTK3, libcurl, OpenSSL

## 3. 安装

### 3.1 安装依赖

```bash
sudo apt-get update
sudo apt-get install -y cmake gcc libgtk-3-dev libcurl4-openssl-dev libssl-dev pkg-config
```

### 3.2 编译 SDK

```bash
cd /workspace
./build_sdk.sh
```

### 3.3 编译上位机程序

```bash
cd /workspace
./build_all.sh
```

## 5. 操作步骤

### 5.1 启动程序

```bash
cd /workspace/ipad_host_app/build
./ipad_host_app
```

### 5.2 调用 API

1. **选择 API**
   - 在左侧「可用 API」树中展开分类
   - 点击要调用的 API 名称

2. **输入参数**
   - 右侧会自动显示该 API 的参数输入框
   - 参数类型包括：
     - 字符串输入框
     - 数值输入框
     - 下拉选择框（枚举值）
     - 复选框（布尔值）

3. **参数验证**
   - 程序会自动验证必填参数是否已填写
   - 格式验证（如 ICCID 为 19-20 位数字）
   - 参数格式不正确会显示红色警告

4. **调用 API**
   - 点击「调用 API」按钮
   - 等待响应（通常 1-3 秒）
   - 查看底部日志区的响应结果

### 5.3 手动响应参数注入（调试功能）

用于测试异常场景和错误处理：

1. 勾选「启用手动响应参数注入」
2. 从下拉框选择要返回的错误码
3. 调用 API 时不会真正执行，直接返回设定的错误

### 5.4 导出日志

1. 点击「导出日志」按钮
2. 选择保存路径和文件名
3. 日志文件包含所有 API 调用记录和响应信息

## 6. API 分类说明

### 6.1 信息获取类

| API 名称 | 功能描述 | 参数 |
|---------|---------|------|
| 获取证书信息 | 获取 eUICC 证书（EUM 和 eUICC 证书） | 无 |
| 获取所有 Profile 信息 | 获取已安装的所有 Profile 信息 | 无 |
| 获取 eIM 配置 | 获取 eIM 配置信息 | 无 |
| 获取 euiccInfo1 | 获取 eUICC 基本信息 | 无 |
| 获取 euiccInfo2 | 获取 eUICC 详细信息 | 无 |
| 获取 EID | 获取 eUICC EID（十六进制字符串） | 无 |

### 6.2 Fallback 机制

| API 名称 | 功能描述 | 参数 |
|---------|---------|------|
| 执行 Fallback | 启用 Fallback Profile | 无 |
| 从 Fallback 返回 | 恢复正常 Profile 操作 | 无 |
| 报告回滚结果 | 向 eUICC 报告 Fallback Profile 回滚测试结果 | 回滚结果结构 |

### 6.3 应急功能

| API 名称 | 功能描述 | 参数 |
|---------|---------|------|
| 启用应急 Profile | 启用应急 Profile | 无 |
| 禁用应急 Profile | 禁用应急 Profile | 无 |

### 6.4 通知处理

| API 名称 | 功能描述 | 参数 |
|---------|---------|------|
| 发送所有通知 | 发送所有待处理的 eUICC 通知到 SM-DP+ | 无 |
| 发送单个通知 | 发送指定序列号的单个通知 | remove_after_send, smdp_address, seq_number |
| 移除通知 | 移除指定序列号的通知 | sequence_number |
| 移除所有通知 | 移除所有待处理的通知 | 无 |

## 7. 常见问题

### Q1: 程序启动失败

**问题**: 提示找不到共享库

**解决**:
```bash
# 设置 LD_LIBRARY_PATH
export LD_LIBRARY_PATH=/workspace/ipa_sdk_pc/dist/lib:$LD_LIBRARY_PATH
./ipad_host_app
```

### Q2: SDK 初始化失败

**问题**: 日志显示「IPA 初始化失败」

**解决**:
1. 检查是否连接到 eUICC 设备
2. 确认设备驱动已正确安装
3. 查看系统日志：`dmesg | grep USB`

### Q3: API 调用超时

**问题**: 调用 API 时长时间无响应

**解决**:
1. 检查网络连接（如调用涉及 SM-DP+ 通信）
2. 增大超时时间设置
3. 查看防火墙配置

### Q4: 参数验证失败

**问题**: 输入的参数被判定为无效

**解决**:
- **ICCID**: 必须是 19-20 位纯数字
- **SM-DP+ 地址**: 必须是有效的 URL 格式
- **确认码**: 4-8 位字母数字组合
